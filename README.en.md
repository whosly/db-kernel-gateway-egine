# db-kernel-gateway-egine

#### Description
多协议数据库网关引擎，是一个数据库协议代理系统，旨在为不同的数据库系统提供统一的访问接口。该项目基于SQL解析器原理，使用Java 17和Spring Boot框架开发，支持多种数据库协议（如MySQL、PostgreSQL、Oracle等）的客户端连接，并能将这些连接代理到相应的后端数据库。
